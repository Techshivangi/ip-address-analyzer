# IP Address Analyzer

## Overview
This project is a simple Python-based IP Address Analyzer that validates IPv4 addresses and provides useful network information.

## Features
- Checks if an IP address is valid
- Identifies Private or Public IP
- Classifies IP into Class A, B, C, D, or E

## How it works
- Takes user input
- Splits IP into 4 parts
- Validates each part (0–255 range)
- Detects network type and class

## Example
Input:
192.168.1.10

Output:
Valid IP Address  
Type: Private IP Address  
Class: C

## Tech Used
- Python (basic logic, conditionals, loops)

## Purpose
This project was created as a beginner cybersecurity exercise to understand IP addressing concepts.
